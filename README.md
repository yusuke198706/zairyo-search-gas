## 🎯 このプロジェクトについて

### 解決した課題
配電工事では工事パターンごとに必要な材料が決まっています。従来は：

- 📋 工事パターンから材料を手動でリストアップ → **抜け漏れが多発**
- 🔢 複数パターンの材料を合算する作業が手間 → **集計ミスが起きやすい**
- 📄 材料表を紙で管理 → **現場でのリアルタイム確認が困難**

### 実現したこと
**「パターンと数量を選ぶだけで必要材料が自動集計される」** システムを構築。
複数カテゴリ・複数パターンの材料を瞬時に合算して一覧表示。

---

## ✨ 主な機能

### 📋 カテゴリ別材料検索
以下の6カテゴリに対応：
- 高圧 / 低圧 / 単相変台 / 灯動変台 / 架空地線 / 支線

### ➕ 複数パターンの追加
同一カテゴリで複数パターンを追加して同時に検索可能

### 🔢 自動集計
複数パターンの材料を自動で合算して重複なく一覧表示

### 📊 材料グループ分類
カバー・ボルト・バンド・タイ・アーム・碍子など種別ごとに自動ソート

### 🖨️ 印刷対応
検索結果をそのまま印刷可能（印刷時は不要なUIを自動非表示）

### 📥 CSV出力
検索結果をCSVファイルとしてダウンロード可能

### 📜 検索履歴
過去の検索内容と結果をローカルに保存・表示

## 🚀 セットアップ方法

### 1. スプレッドシートの準備
上記の形式で `シート2` に材料マスタを入力してください。

### 2. GASへのコード貼り付け
1. スプレッドシートを開く
2. `拡張機能` → `Apps Script` を開く
3. `コード.gs` と `index.html` を貼り付けて保存

### 3. Webアプリとしてデプロイ
1. `デプロイ` → `新しいデプロイ` をクリック
2. 種類：`ウェブアプリ`
3. アクセス権限：`全員`
4. デプロイ後に発行されるURLにアクセスして完了

## 💡 工夫したポイント

### 材料の自動グループ分類
材料名のキーワードから種別を自動判定してソート：

```javascript
function getMaterialGroup(name) {
  const groups = [
    { keyword: 'カバー', group: '01_カバー' },
    { keyword: 'ボルト', group: '02_ボルト' },
    { keyword: 'バンド', group: '03_バンド' },
    // ...
  ];
  for (const item of groups) {
    if (name.includes(item.keyword)) return item.group;
  }
  return '99_その他';
}
```

### 複数パターンの集計
同じ材料が複数パターンに含まれる場合、数量を自動合算：

```javascript
function aggregateResults(results) {
  const aggregatedResults = {};
  results.forEach(({ material, quantity }) => {
    aggregatedResults[material] = (aggregatedResults[material] || 0) + quantity;
  });
  return aggregatedResults;
}
```
